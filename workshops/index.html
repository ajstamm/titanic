Icebergs Ahead: An introduction to R
This is the first workshop in the Titanic Introduction to R (TITR) series. If you know enough about R to be interested in this workshop, you are probably prepared to take it.
By the the end of the workshop, participants will learn:
  • basic data management techniques
  • simple statistical tests
  • primary data types, vectors, and data.frames
  • practical skills such as filtering data
  • basics of RStudio

R-requisites: None.

Stats-requisites: You should understand the basics of Null Hypothesis Signifiance Testing (NHST).

Women and Children First: Regression in R
This is the second workshop in the TITR series. This assumes the participant has completed the first course and has a passing familiarity with R (or is an experienced programmer/fast learner).
Reviews and builds on the data management skills introduced in "Icebergs Ahead". Participants use these skills to build linear and logistic regression models.

R-requisites: Icebergs Ahead! or comparable experience using R

Stats-requisites: You should understand the concepts behind linear and logistic regression

Data Management in R
This is the third workshop in the TITR series. This assumes the participant has completed the first course or has equivalent knowledge of R.
Builds on the data management skills introduced in "Icebergs Ahead" by introducing an entirely new way to program in R using tidyverse.

R-requisites: Experience using R

Stats-requisites: None.

These materials are free to use outside of this workshop. If you use these materials in your own workshop, please let us know, and have a great time!

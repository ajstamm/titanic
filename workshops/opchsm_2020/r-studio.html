RStudio IDE

RStudio windows

Ways to run code

REPL Window
  • Press Enter
  • Highly interactive
  • History saved automatically
  • Best for exploration / experimentation
Editor Window
  • Press Control+Enter
  • Less interactive
  • History saved when you save the file
  • Best for documented / reproducible work

Using an RStudio project

Save this ZIP file to your desktop and unzip it. Click on the file "ocphsm.Rproj". This will open your project in RStudio.

Projects use relative paths, so they are an easy way to share data and code. To access a file in the project's root folder, just call the filename without a folder. If you prefer to place your data in a "data" folder, use "/" to separate folder and file names (NOT "\" used in Windows).


# example filename only
my_filename <- "my_file.csv"
# example filepath and filename
my_filename <- "data/my_file.csv"

Installing packages

You will need two packages today:

  • tidyverse: a consellation of data management packages
  • openxlsx: an interpreter to import Excel files into R

To import a file, click "Tools" > "Install packages ..." and type the package names above into the text box under "Packages". Keep "Install dependencies" checked and click "Install", then wait a couple minutes while several packages install.

After the packages are installed, try loading them:


library(tidyverse)
library(openxlsx)

Other Features

  • Lists files - click to open
  • Lists data objects and functions
  • Provides help information look-up
  • Organizes plots & Graphs
  • Converts reports from Markdown to HTML, PDF, Word

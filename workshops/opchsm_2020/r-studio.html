About RStudio

  • Available on Linux, Mac, and Windows
  • Requires R to run; install R first
  • Cost == $0 (AGPL3)
  • https://www.rstudio.com/products/rstudio/

RStudio IDE

RStudio windows

Ways to run code

REPL Window
  • Press Enter
  • Highly interactive
  • History saved automatically
  • Best for exploration / experimentation
Editor Window
  • Press Control+Enter
  • Less interactive
  • History saved when you save the file
  • Best for documented / reproducible work

Using an RStudio project

Save this ZIP file to your desktop and unzip it. Click on the file "ocphsm.Rproj".

Projects use relative paths. To access a file in the project's root folder, call the filename without a folder.

Note: If you use a "data" folder, separate folder and file names with "/" (not "\").


# example filename only
my_filename <- "my_file.csv"
# example filepath and filename
my_filename <- "data/my_file.csv"

Installing packages

You will need two packages today:

  • tidyverse: a consellation of data management packages
  • openxlsx: an interpreter to import Excel files into R

To import a file:

  1. Click "Tools" > "Install packages ..."
  2. Type package names into the text box under "Packages"
  3. Keep "Install dependencies" checked
  4. Click "Install" and wait

Load packages using libaray().

Other Features

  • Lists files - click to open
  • Lists data objects and functions
  • Provides help information look-up
  • Organizes plots & Graphs
  • Converts reports from Markdown to HTML, PDF, Word

RStudio is the most popular IDE for R, but not the only one. You can also run R in SAS.

2 Ways To Run

REPL Window
  • Press Enter
  • Highly interactive
  • History saved automatically
  • Best for exploration / experimentation
Editor Window
  • Press Control+Enter
  • Less interactive
  • History saved when you save the file
  • Best for documented / reproducible work

Other Features

  • Data objects and functions
  • Help information
  • Plots & Graphs
  • File lists - click to open file
  • Convert reports from Markdown to HTML, PDF, Word

RStudio is the most popular IDE for R, but not the only one. You can also run R in SAS.

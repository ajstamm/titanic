2 Ways To Run

REPL Window
  • Hit Enter
  • Highly interactive
  • History saved automatically
  • Best for exploration / experimentation
Editor Window
  • Hit Control+Enter
  • Less interactive
  • History saved when you save the file
  • Best for documented / reproducible work
